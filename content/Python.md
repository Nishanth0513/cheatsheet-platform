## Variables and Data Types

```python
# Strings
name = "Python"

# Numbers
age = 30
pi = 3.14159

# Boolean
is_awesome = True

# Lists
languages = ["Python", "JavaScript", "Go"]

# Dictionaries
user = {"name": "John", "age": 28}
```

## Control Flow

```python
# If statements
if age > 18:
    print("Adult")
elif age > 12:
    print("Teenager")
else:
    print("Child")
    
# Loops
for language in languages:
    print(language)
    
while count > 0:
    print(count)
    count -= 1
```
